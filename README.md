# D3D12-Wallhack with Imgui Overlay
d3d12 wallhack, dx12 wallhack, imgui, overlay

1. Compile Dll
2. Run game in window mode or borderless 
3. Inject Dll
4. Press Insert for menu
5. Bruteforce Stride hash and filter rendertarget ect. to find models  
6. Close menu to save settings
